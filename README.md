# Tu-154B2 CE (Community Edition)

## Overview
This is a continuation of the Tu-154B2 project by Masterskaya which itself is based on the original Tu-154M by Felis.
The intention is to fully overhaul the aircraft visuals and systems to bring it up to date with X-Plane 12.

## Installation
- Visit the Release page for the latest version of the Tu-154B2 CE.
- Unpack the downloaded archive into your X-Plane 12's Aircraft directory.

## What about the Tu-154M? Is it abandoned?

Essentially... yes. The B2 variant is now considered the primary variant. After overhaul it'll serve
as a base for M variant.
